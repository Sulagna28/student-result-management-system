# student-result-management-system
A beginner-level SQL project built using MySQL Workbench to manage student results, subjects, and marks.# Student Result Management System

## Description
This is a beginner-level SQL project developed using MySQL Workbench.  
It manages student results, subjects, and marks efficiently.

## Features
- Student details management
- Subject-wise marks storage
- Result calculation
- Structured relational database

## Tools Used
- MySQL
- MySQL Workbench

## How to Use
1. Download the student_result_project.sql file
2. Open MySQL Workbench
3. Create a new schema
4. Import the SQL file
5. Execute queries
